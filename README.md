# Gamification-based Aspect-Based Sentiment Analysis for Shopee Reviews using Deep Learning

This repository contains the implementation developed for the undergraduate thesis entitled

"Gamification-based Aspect-Based Sentiment Analysis of Shopee User Reviews Using Deep Learning Models"

---

## Overview

This study proposes an Aspect-Based Sentiment Analysis (ABSA) framework to analyse user perceptions toward gamification elements implemented in the Shopee mobile application.

The proposed framework integrates

- Dictionary-based Aspect Identification
- Manual Sentiment Annotation
- Five Deep Learning Models
- Purchase Intention Analysis
- Gamification Pyramid

---

## Models

- TD-LSTM
- Bi-LSTM
- GRU
- Attention Bi-LSTM
- IndoBERT
